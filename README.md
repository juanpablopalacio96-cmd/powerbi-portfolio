# Power BI Portfolio

This repository showcases Power BI dashboards focused on warehouse operations, picking productivity, quality control, and equipment utilization.

The projects are based on real operational use cases, with data anonymized and designed to support decision-making in logistics and industrial environments.

## Projects

- Warehouse Picking Productivity Dashboard
- Picking Errors & Quality Analysis Dashboard
- Forklift Utilization Productivity Dashboard

Each project includes dashboard screenshots, business context, key KPIs, and insights.
