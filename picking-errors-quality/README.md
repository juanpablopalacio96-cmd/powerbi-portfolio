## Picking Errors & Quality Analysis Dashboard

### Context
Dashboard developed to analyze picking errors and product damage, helping improve quality control and reduce operational losses.

### Business Questions
- What is the error rate compared to total picked units?
- Which operators, products, or picking zones generate more errors?
- How does error behavior evolve over time?

### KPIs
- Total picking errors
- Error rate (% errors vs picked units)
- Damaged units
- Errors by operator, product, and picking area

### Tools
- Power BI
- DAX
- Excel

### Outcome
The dashboard supports quality improvement initiatives by identifying error sources and enabling targeted corrective actions at operator, product, or process level.
