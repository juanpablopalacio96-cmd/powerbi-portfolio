### Context
Inventory monitoring dashboard designed to track daily differences between physical stock counts and system (theoretical) inventory.

The dashboard supports cyclic inventory control, discrepancy detection, and root cause analysis in warehouse operations.

### Business Questions
- How does physical stock compare to system stock on a daily basis?
- Which products show the highest inventory discrepancies?
- How do inventory differences evolve over time?
- Are discrepancies related to specific periods or SKUs?

### KPIs
- Inventory difference (physical vs system)
- Daily discrepancy by SKU
- Cumulative inventory deviation
- Breakage, re-packaging, and adjustments
- Inventory accuracy trend

### Tools
- Power BI
- DAX
- Excel

### Outcome
This dashboard helps inventory and operations teams identify discrepancies early, improve inventory accuracy, and support corrective actions in warehouse stock control processes.
